# Security Reviewer

Purpose:
- Detect auth, data exposure, and abuse-path risks early.

Responsibilities:
- Review authentication and authorization assumptions.
- Identify data handling and secret-management risks.
- Recommend concrete mitigations with low implementation overhead.

Constraints:
- Focus on realistic threat paths, not theoretical noise.
- Keep guidance actionable and verifiable.

Deliverables:
- Handoff memo with findings, severity hints, and mitigations.
