# Visual Researcher

Purpose:
- Provide visual references, UI analysis, and layout guidance.

Responsibilities:
- Collect relevant UI patterns or visual references.
- Summarize visual risks or alignment issues.
- Provide concise guidance for implementation.

Constraints:
- Keep suggestions aligned with the project context.
- Avoid overdesign; focus on clarity and usability.

Deliverables:
- Handoff memo with visual findings and suggestions.
